Détection d'anomalies dans les données de consommation énergétique¶

Ce projet a pour objectif de détecter des anomalies dans les données de consommation énergétique à l'aide d'algorithmes d'apprentissage automatique. Nous utiliserons des modèles tels que Isolation Forest et One-Class SVM pour la détection d'anomalies. Le but est d'identifier des comportements inhabituels de consommation d'énergie qui pourraient indiquer des inefficacités ou des problèmes potentiels.

# Importation des bibliothèques nécessaires
import pandas as pd

Comprendre la problèmatique :¶

Contexte :¶

Dans un contexte où la consommation énergétique augmente, il est crucial pour les entreprises et les collectivités de surveiller leur consommation d'énergie pour détecter des anomalies qui pourraient indiquer des problèmes techniques ou des usages inefficaces. Les anomalies peuvent être dues à des pannes d'équipement, à des comportements inhabituels des consommateurs ou à des variations saisonnières non anticipées.¶

Problématique¶

Comment détecter efficacement les anomalies dans les données de consommation énergétique d'une entreprise au fil du temps afin de minimiser les coûts d'exploitation et d'optimiser la consommation d'énergie ?¶

Étape 5 : Détection des anomalies avec Isolation Forest¶

L'algorithme Isolation Forest est utilisé pour détecter des points de données qui semblent être des "outliers" par rapport aux autres. Nous allons entraîner ce modèle sur la consommation d'énergie et identifier les points qui semblent anormaux.

In [ ]:
# Créer un modèle Isolation Forest
model = IsolationForest(contamination=0.02)  # Ajustez la contamination selon vos besoins

# Ajuster le modèle
data['Anomaly'] = model.fit_predict(data[['EnergyConsumption']])

# Visualiser les anomalies
plt.figure(figsize=(12, 6))

# Tracer la consommation d'énergie
plt.plot(data.index, data['EnergyConsumption'], label='Consommation d\'énergie')

# Tracer les anomalies
plt.scatter(data.index[data['Anomaly'] == -1], data['EnergyConsumption'][data['Anomaly'] == -1], color='red', label='Anomalies')

# Personnaliser le graphique
plt.title('Détection d\'Anomalies dans la Consommation d\'Énergie')
plt.xlabel('Date')
plt.ylabel('Consommation (kWh)')
plt.legend()
plt.show()

Étape 6 : (Supplémentaire) Détection des anomalies avec One-Class SVM¶

L'algorithme One-Class SVM est une autre méthode de détection d'anomalies. Il fonctionne en apprenant les frontières des données normales et en marquant celles qui tombent en dehors de ces frontières comme anormales.

In [ ]:
from sklearn.svm import OneClassSVM

# Create OneClassSVM model
svm_model = OneClassSVM(nu=0.05, kernel="rbf", gamma=0.1)

# Fit the model
data['Anomaly_SVM'] = svm_model.fit_predict(data[['EnergyConsumption']])

# Visualize anomalies detected by SVM
plt.figure(figsize=(12, 6))
plt.plot(data.index, data['EnergyConsumption'], label='Energy Consumption')
plt.scatter(data.index[data['Anomaly_SVM'] == -1], 
            data['EnergyConsumption'][data['Anomaly_SVM'] == -1], 
            color='orange', label='Anomalies by SVM', marker='o')
plt.title('Anomaly Detection using One-Class SVM')
plt.xlabel('Date')
plt.ylabel('Energy Consumption (kWh)')
plt.legend()
plt.grid(True)
plt.show()
